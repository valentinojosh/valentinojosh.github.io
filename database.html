Basketball Statistics Database

For my Database Design and Implementation course, I created a functional database in SQL utilizing MySQL Workbench running a local server. This project included the design of the ERD, creation of tables and their relationships in SQL, insertion of data, samples of complex queries, and connection via web application. All the assignments, descriptions, and sample queries related to this project are on my GitHub.
